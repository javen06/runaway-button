# Runaway Button

A playful single-page JavaScript experiment where the **No** button moves away
from the pointer while the **Yes** button reveals a response.

## Live Demo

https://javen06.github.io/runaway-button/

## Features

- Pointer and touch-friendly evasive button interaction
- Responsive layout for desktop and mobile screens
- Lightweight implementation with no build tools or dependencies
- Keyboard-accessible buttons and reduced-motion support

## Tech Stack

- HTML
- CSS
- JavaScript

## Run Locally

```bash
git clone https://github.com/javen06/runaway-button.git
cd runaway-button
```

Open `index.html` in a browser or use a static development server such as the
VS Code Live Server extension.
